# Link-Bypasser-Bot

a Telegram Bot that can Bypass the Ad Links

---

## Variables

- `TOKEN` Your bot token from @BotFather

---


## Commands to set in bot

```
start - Welcome Message with supported sites

af - adfly

gp - gplinks

dl - droplink

lv - linkvertise

md - mdisk

rl - rocklinks

pd - pixeldrain

wt - wetransfer

mu - megaup

gd - Google Drive Look-Alike

ot - others

ou - ouo
```
